# MessAuto-Electron

> 一个基于 Electron 的 macOS 桌面应用程序，用于自动提取短信验证码并实现自动粘贴和回车功能。

## 功能特性

- 🔍 **自动监控**: 实时监控 macOS Messages 数据库，自动检测新的短信验证码
- 📋 **智能提取**: 使用先进的正则表达式和关键词匹配技术，准确提取验证码
- ⚡ **自动操作**: 支持自动粘贴验证码到活动窗口，并可选择自动按回车
- 🎯 **系统托盘**: 在系统托盘中运行，提供便捷的菜单操作
- 🔐 **权限管理**: 智能检测和引导用户授予必要的系统权限
- 🌐 **多语言**: 支持中文和英文界面
- 🚀 **开机自启**: 可设置为系统启动时自动运行

## 系统要求

- **操作系统**: macOS 10.14+ (Mojave 或更高版本)
- **架构**: 支持 Intel (x64) 和 Apple Silicon (arm64)
- **权限**: 需要完全磁盘访问权限和辅助功能权限

## 快速开始

### 安装

1. 从 [Releases](https://github.com/your-repo/messauto-electron/releases) 页面下载最新版本
2. 双击 `.dmg` 文件进行安装
3. 将 MessAuto 拖拽到应用程序文件夹

### 首次使用

1. **启动应用**: 双击应用图标启动 MessAuto
2. **授予权限**: 
   - 系统会提示需要完全磁盘访问权限，点击"授予权限"按钮
   - 在系统偏好设置中找到 MessAuto 并启用权限
   - 同样操作授予辅助功能权限
3. **配置功能**: 在应用界面中启用所需的功能（自动粘贴、自动回车等）
4. **开始使用**: 应用会自动监控短信验证码，无需手动操作

## 使用指南

### 权限配置

#### 完全磁盘访问权限
1. 打开"系统偏好设置" > "安全性与隐私" > "隐私"
2. 选择"完全磁盘访问"
3. 点击锁图标并输入密码
4. 点击"+"按钮，选择 MessAuto 应用
5. 确保 MessAuto 旁边的复选框已勾选

#### 辅助功能权限
1. 打开"系统偏好设置" > "安全性与隐私" > "隐私"
2. 选择"辅助功能"
3. 点击锁图标并输入密码
4. 点击"+"按钮，选择 MessAuto 应用
5. 确保 MessAuto 旁边的复选框已勾选

### 功能设置

#### 自动粘贴
- **说明**: 检测到验证码后自动粘贴到当前活动窗口
- **使用**: 在设置中启用"自动粘贴"功能
- **注意**: 确保目标输入框处于焦点状态

#### 自动回车
- **说明**: 粘贴验证码后自动按回车键
- **依赖**: 需要先启用"自动粘贴"功能
- **使用**: 在设置中启用"自动回车"功能

#### 开机自启动
- **说明**: 系统启动时自动运行 MessAuto
- **使用**: 在设置中启用"登录时启动"功能

### 托盘菜单

点击系统托盘中的 MessAuto 图标可以快速操作：

- ✅/❌ **自动粘贴**: 快速开启/关闭自动粘贴功能
- ✅/❌ **自动回车**: 快速开启/关闭自动回车功能
- 📁 **隐藏图标**: 临时或永久隐藏托盘图标
- ✅/❌ **登录时启动**: 开启/关闭开机自启动
- ⚙️ **设置**: 打开设置窗口
- ❌ **退出**: 完全退出应用

## 验证码支持

### 支持的格式
- **纯数字**: 4-8 位数字验证码 (如: 1234, 123456)
- **字母数字**: 4-8 位字母数字组合 (如: A1B2, XY123Z)
- **带关键词**: 包含"验证码"、"code"等关键词的消息

### 支持的语言
- **中文**: 验证码、驗證碼
- **英文**: verification code, verification, code, OTP
- **韩文**: 인증

### 检测逻辑
1. **关键词检测**: 短信内容必须包含验证码相关关键词
2. **格式匹配**: 使用正则表达式匹配验证码格式
3. **置信度计算**: 根据上下文和格式计算可信度
4. **最优选择**: 选择置信度最高的验证码

## 隐私与安全

- **本地处理**: 所有数据处理均在本地进行，不会上传到任何服务器
- **权限最小化**: 仅请求必要的系统权限
- **数据安全**: 不会存储或记录短信内容，仅提取验证码
- **开源透明**: 源代码公开，可审查安全性

## 故障排除

### 常见问题

#### 应用无法检测验证码
1. 确认已授予完全磁盘访问权限
2. 检查 Messages 应用是否正常工作
3. 验证短信内容是否包含支持的关键词
4. 重启 MessAuto 应用

#### 自动粘贴不工作
1. 确认已授予辅助功能权限
2. 检查目标输入框是否处于焦点状态
3. 尝试使用"测试粘贴"功能验证
4. 重启 MessAuto 应用

#### 托盘图标消失
1. 检查"隐藏图标"设置是否被启用
2. 重启 MessAuto 应用
3. 在应用程序文件夹中重新启动

### 日志和调试

如需技术支持，可以：
1. 在设置中启用调试模式
2. 查看控制台日志
3. 联系技术支持并提供日志信息
