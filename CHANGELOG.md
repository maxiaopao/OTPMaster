# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial project setup
- Core SMS monitoring functionality
- Automatic verification code extraction
- Auto-paste and auto-enter features
- System tray integration
- Permission management system
- Multi-language support (Chinese, English)
- Vue 3 + TypeScript frontend
- Electron main process implementation

### Features
- Real-time SMS database monitoring
- Intelligent verification code extraction using regex patterns
- Automatic clipboard operations
- System tray menu for quick access
- Permission status monitoring
- Launch at login option
- Verification code history
- Modern Vue 3 interface with Composition API

### Technical
- TypeScript throughout the project
- Electron + Vue 3 + Vite architecture
- Better-sqlite3 for database access
- RobotJS for automation
- Chokidar for file watching
- Vue I18n for internationalization
- Pinia for state management
- ESLint + Prettier for code quality
